Model pudełkowy CSS (Box Model)

  1. Co to jest model pudełkowy?
    Model pudełkowy opisuje sposób, w jaki przeglądarka renderuje elementy HTML jako prostokątne pudełka.
  2. Z jakich części składa się pudełko?
    1. content (treść)
    2. padding (wewnętrzny margines)
    3. border (obramowanie)
    4. margin (zewnętrzny margines)
  3. Kolejność warstw (od środka): content → padding → border → margin
  4. Wzór na całkowitą szerokość elementu:
    width + padding-left + padding-right + border-left + border-right + margin-left + margin-right
